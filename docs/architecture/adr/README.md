## Architecture Decision Records

We gebruiken ADR's (Architecture Decision Records) om belangrijke technische beslissingen
vast te leggen. Elke ADR beschrijft de context, overwegingen, en het besluit.

### Formaat

We volgen het [Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)
formaat: Status, Context, Decision, Consequences.
